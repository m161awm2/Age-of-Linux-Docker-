# ⚔️ 에이지 오브 리눅스: 전직의 시대 (v2.0)

에이지 오브 리눅스는 터미널 환경에서 즐기는 **실시간 전략 디펜스 게임**입니다.  
골드를 관리하여 병사를 전직시키고, 훈련을 통해 군대를 강화하여 **적 기지(500 HP)**를 파괴하세요!

# 🎮 조작법 (Controls)

| 키      | 액션           | 설명 |
|---------|----------------|------|
| [1]     | 보병 생산      | 전선의 핵심입니다. (4G ~ 6G) |
| [2]     | 궁수 생산      | 후방 지원 유닛입니다. (6G ~ 7G) |
| [3]     | 기병 생산      | 강력한 돌격 유닛입니다. (14G ~ 20G) |
| [4]     | 스페셜 메뉴    | 펜리르, 로닌 등 특수 유닛 해금 및 생산 창을 엽니다. |
| [5]     | 전직 메뉴      | 보병 → 궁수 → 기병 순으로 전직 선택창을 호출합니다. |
| [6~8]   | 선택 / 해금   | 전직 및 스페셜 메뉴 내에서 유닛을 결정합니다. |
| [9]     | 군대 훈련      | 전 유닛 HP/DMG 20% 복리 상승. 비용은 레벨마다 증가합니다. |
| [Q]     | 게임 종료      | 터미널로 즉시 돌아갑니다. |

# 💡 승리를 위한 핵심 전략

### ⚔️ 드라군(D)의 위치 선정
- 거리가 **1.5 이하**로 좁혀지면 칼을 뽑습니다.  
- 공격력이 **12로 상승**, 쿨다운이 **1초로 감소**, 화력이 폭증합니다.

### 🔥 불화살(F)과 훈련
- 퍼센트 데미지는 **훈련 레벨([9] 키)**에 비례해 강화됩니다.  
- 고체력 유닛(스파르타, 전차) 대응 필수.

### 🤖 AI의 역습
- AI는 자원 소모 없이 일정 시간마다 전직합니다.  
- **적 기지 체력이 50씩 감소할 때마다 AI에게 20G 보급**.

### 🐺 펜리르(L) 카운터 전략
- 적 궁수 라인이 강력할 때 투입하면 원거리 피해 40% 경감, 후방을 효과적으로 파고듭니다.

### 3️⃣ 무한 강화 (Army Level Up)
- AI는 전직은 하지만 **레벨 업([9] 키)**은 하지 않습니다.
